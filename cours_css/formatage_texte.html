La taille

propriété CSS font-size

  • Exprimée en taille absolue: en pixels (px),centimètre, millimètres...
  • Exprimée en taille relative: en pourcentage, "em"ou "ex", cette technique a l'avantage d'être plus souple. Elle s'adapte plus facilemnt aux préférences de taille des visiteurs.
    • XX-small:minuscule ;
    • x-small:très petit ;
    • small: petit ;
    • medium: moyen ;
    • large: grand ;
    • x-large: très grand ;
    • xx-large: gigantesque
    • Uniquement 7 tailles.
    • Préférez les tailles exprimées en em
    • Si vous écrivez 1em, le texte a une taille normale.
    • Si vous voulez grossir le texte, vous pouvez inscrire unevaleur supérieure à 1,comme 1.3em.
    • Si vous voulez réduire le texte, inscrivez une valeur inférieur à 1, comme 0.8em.
